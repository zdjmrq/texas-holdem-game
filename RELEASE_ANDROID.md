# Texas Hold'em Game - Android 预览版

## v0.1.0-alpha

首个 Android APK 安装包，由 Web 版本通过 WebView 打包为安卓应用。

### 下载

APK 安装包请从 [Releases](https://github.com/zdjmrq/texas-holdem-game/releases) 页面下载。

### 已知问题

- **界面适配尚未完成**：当前 UI 布局直接继承自 Web 版本，尚未针对手机屏幕进行适配优化，界面较为混乱
- 部分交互体验待优化

### 安装说明

1. 下载 APK 文件
2. Android 设备：设置 → 安全 → 允许安装未知来源应用
3. 点击 APK 文件安装

### 技术说明

- 打包方式：Cordova/Capacitor WebView
- 游戏引擎：HTML5 + JavaScript
- 当前版本仅供开发测试，非正式发布版本
